# Skill Catalog Template

## Metadata
- Path:
- Owner:
- Last updated:
- Classification: `template-owned skeleton`

## Purpose
- Provide the reusable structure for repository skill catalogs.
- Keep catalog structure reusable while skill registration remains repository-instance data.

## Template Rules
- The catalog structure is reusable.
- Skill rows, paths, intents, and statuses are repository-instance data.
- Each repository instance catalog should align with `project_profile.yaml.skills.*`.

## Recommended Entry Fields
- skill name
- local path
- intent
- status
