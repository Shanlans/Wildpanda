# Procedure Catalog Template

## Metadata
- Path:
- Owner:
- Last updated:
- Classification: `template-owned skeleton`

## Purpose
- Provide the reusable structure for repository procedure catalogs.
- Separate reusable catalog structure from repository-instance procedure entries.

## Template Rules
- The catalog structure is reusable.
- Procedure rows, names, scopes, and document paths are repository-instance data.
- Each repository instance catalog should reference `PROCEDURE_TEMPLATE.md` as the procedure-structure template source.

## Recommended Entry Fields
- procedure id
- name
- scope
- instance document path
- status
