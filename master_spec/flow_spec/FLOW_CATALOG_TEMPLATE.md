# Flow Catalog Template

## Metadata
- Path:
- Owner:
- Last updated:
- Classification: `template-owned skeleton`

## Purpose
- Provide the reusable structure for repository flow catalogs.
- Separate reusable catalog structure from repository-instance flow entries.

## Template Rules
- The catalog structure is reusable.
- Flow rows, names, scopes, and document paths are repository-instance data.
- Each repository instance catalog should reference `FLOW_TEMPLATE.md` as the flow-structure template source.

## Recommended Entry Fields
- flow id
- name
- scope
- instance document path
- status
